# Go Cheatsheets

[![License](https://img.shields.io/github/license/GoEcosystem/go-cheatsheets.svg)](https://github.com/GoEcosystem/go-cheatsheets/blob/main/LICENSE)

A collection of Go programming language cheatsheets and quick reference guides for developers in the GoEcosystem organization.

## Contents

### Basics
- [Basic Syntax](basics/basic-syntax.md) - Variables, functions, control flow, and basic syntax
- [Data Structures](basics/data-structures.md) - Arrays, slices, maps, and structs

### Advanced
- [Error Handling](advanced/error-handling.md) - Working with errors and custom error types
- [Concurrency](concurrency/concurrency.md) - Goroutines, channels, and synchronization primitives

### Standard Library
- [Standard Library](standard-library/standard-library.md) - Common packages and functions from the standard library

### Development
- [Testing](tooling/testing.md) - Unit testing, benchmarking, and mocking

## Purpose

These cheatsheets serve as quick reference guides for Go programmers working on GoEcosystem projects. They cover:

- Core language features and syntax
- Standard library functions and methods
- Best practices and common patterns
- Quick examples for common tasks

## How to Use

Each cheatsheet is a standalone document focusing on a specific topic. They are designed to be concise, providing just enough information to remind you of syntax or patterns without having to search through extensive documentation.
